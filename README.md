# PharmaGo
📦 PharmaGo | Fullstack Medical Ecosystem  PharmaGo is a high-load platform bridging the gap between customers, pharmacies, and depots.  🚀 Tech: .NET 8, PostgreSQL, React, React Native &amp; Telegram Bot. 🔹 Features: Intelligent generics search, real-time stock mapping, SignalR notifications, and dedicated inventory management for pharmacists.
